# staking-contract
Staking contract using in Evrynet Proof-of-Stake model

## Setup
1. Clone this repo
2. `npm ci`

## Compilation
`yarn run buidler compile`

## Linting
```
npm run solhint
npm run prettier
```

## Testing with Buidler
`yarn run buidler test`

## Testing with coverage
`yarn run buidler coverage --config ./buidler-coverage.config.js`
